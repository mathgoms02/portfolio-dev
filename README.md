# 🚀 Matheus Gomes • Portfolio

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minha trajetória, habilidades técnicas e os principais projetos em que venho trabalhando, com foco em desenvolvimento Fullstack e Inteligência Artificial.

## O Projeto

O design foi construído com uma estética inspirada no estilo _Cyberpunk/Minimalista_, utilizando uma paleta de cores contrastantes sobre um fundo escuro para destacar as informações principais.

### Funcionalidades:

- **Design Responsivo:** Adaptado para dispositivos móveis, incluindo um menu hambúrguer interativo com animações.
- **Efeito Typewriter:** Uma apresentação dinâmica das minhas especialidades na seção de abertura.
- **Carrossel de Projetos:** Navegação fluida entre os trabalhos realizados.
- **Formulário de Contato:** Integração funcional para envio de mensagens via Formspree.
- **Scroll Suave:** Navegação otimizada via JavaScript com cálculo de compensação (offset) para a navbar fixa.
